PySSRS
: Biblioteca Python Open Source para integração com Microsoft SQL Server Reporting Services.
Pode ser utilizado em scripts de automação ou em um projeto Django e Flask. 
Sua maior utilidade está nas versões SQL Server 2008 e inferiores, das quais a Microsoft apenas disponibilizava um protocolo SOAP para integração.
* Linguagem: Python
* Protocolo: SOAP
* https://github.com/FRReinert/PySSRS


Sistema Modular de Integração
: Sistema capaz de mapear o banco de dados do ERP, gerar relatórios customizáveis, automatizar serviços, gerenciar notificações de ações, enviar e-mails automáticos e ainda conta com uma integração com Microsoft Teams, Microsoft Flow e tem todos os acessos gerenciados via Active Directory. 
Este sistema está sendo utilizado por diversos setores de uma empresa para estender funcionalidades não presentes no ERP ou que existiam, porém, não atendiam a demanda e necessitavam de integração com o ERP. 
Exemplos de utilização são a geração de etiquetas UDI, expedição de material utilizando serialização, envio de cobranças para clientes de maneira automatizada, impressão de etiquetas térmicas em impressoras remotas, entre outros recursos.
O sistema é internacionalizado e pode gerar relatórios para Excel, PDF ou para e-mail sem precisar de códigos extras, além de possuir navegação por modelos de dados utilizando Foreign Keys. 
 * Backend: Python, Django, DRF, 
* Frontend: NodeJS, VueJS, Vuetify
* Distribuíção: Ubuntu Server, Nginx
* Escalabilidade: Docker, Docker-Compose, RabbitMQ, Celery, gUnicorn 
* Banco de Dados: MS SQL Server
* Versionamento: Git e Github (Repositório de código fechado)
